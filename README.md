# CryptoSim — симулятор криптовалютного рынка

Веб-приложение для симуляции торговли вымышленными криптовалютами в реальном времени. Учебный проект по курсу «Разработка веб-приложений».

## 🚀 Демо

Приложение развёрнуто на Render: [https://crypto-sim.onrender.com/assets.html](https://crypro-sim.onrender.com/assets.html)

## 📋 Функционал

- **Регистрация и вход** с JWT-аутентификацией
- **Просмотр рынка** — список валют с динамическими ценами
- **Интерактивный график** — SVG, масштабирование колёсиком мыши
- **Торговля** — покупка и продажа валют за USDT
- **Портфель** — отслеживание активов, прибыль/убыток в реальном времени
- **Real-time обновления** — цены меняются через WebSocket
- **Админ-панель** — создание и управление валютами
- **Пополнение баланса** — просмотр рекламы (котик!)

## 🛠️ Технологии

- **Backend:** Go, gRPC, PostgreSQL, Redis
- **Frontend:** HTML/CSS, нативный JavaScript, SVG
- **Инфраструктура:** Docker, Docker Compose, Render
